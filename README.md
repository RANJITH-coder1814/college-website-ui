# 🎓 College Website UI

A simple and responsive **College Website UI** built using HTML and CSS. This project showcases a basic structure of a college website including sections like About, Courses, Faculty, and Contact.

## 🚀 Features

* Responsive design
* Navigation bar
* Hero section with background image
* About section
* Courses display
* Faculty section
* Contact information
* Clean and simple UI

## 🛠️ Technologies Used

* HTML5
* CSS3
